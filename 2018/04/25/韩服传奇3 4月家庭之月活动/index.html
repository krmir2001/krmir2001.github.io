- 活动内容

1. 家庭之月活动

活动日期 : 2018年 4月 25日 更新后 ~ 5月 9日(周三) 更新前

【活动】时空副本的希望之种

原文地址：
http://mir3.mironline.co.kr/news/news_event_view.asp?p=3cf3cfc1&s=3d2c24493d1e0f273d1704963d01e4e33d0ffa05&f=3cfada523d08ef743d01e4e3

活动时间：2018.4.25 ~ 2018.5.9
活动参与方法
  1. 给活动NPC珍妮希望之种可换取希望盒子道具
  2. 交给NPC并登记希望之种12小时后方可获得希望盒子
  3. 可以帮助其他玩家登记的希望之种
  4. 每帮助一次其他玩家可减少40分钟等待时间（最大可帮助15人）
  5. 每帮助一次赠予一个幸运铜币
  6. 希望之种在活动结束后全部删除

NPC对话

活动期间降低时空副本进入等级与减少副本CD时间

希望盒子可开出的道具

新绿祝福 BUFF道具

  • 使用新绿祝福道具可立即赋予角色BUFF
  • BUFF可持续时间为60秒

- 更新内容

1. 新增攻击对象(怪物/玩家)的轮廓线标识功能

2. 道具掉落率调整

  • 怪物等级越低相对低级物品掉落率降低
  • 低于60级的技能书掉落率降低
  • 金币掉率小幅度减少

2. 潘夜神殿更新

  • 现潘夜神殿大厅为基准 下层部分为潘夜神殿 上层部分为古代<远古>潘夜神殿
  • 古代<远古>潘夜神殿属高难度地牢

进入方法

  • 从潘夜神殿3层-西或5层进入 潘夜神殿大厅
  • 潘夜神殿大厅 中央可进入 古代<远古>潘夜神殿

怪物信息请参考日后更新的详细参考页

- 元宝道具更新内容

1. 限量元宝道具开卖

开卖时间 : 2018.04.25(周三) 维护后 ~ 2018.05.09(周三) 维护后

  • 此限量元宝道具有可能以后再次销售

1） 武侠服装箱

  • 销售价格 : 9,900 元宝
  • 商品内容 : 武侠服装箱可开出武侠服装（7天）
  • 商品属性 : 防御+2-2, 魔防 +6-8, 흑막（黑炎防御） +4-4, 舒适 +10, 经验 +5%, 掉落率 +5%

2) 经验瓶(80%)

[销售价格]

  • 经验瓶(80%)(1小时) : 1,100 元宝
  • 经验瓶(80%)(3小时) : 1,900 元宝
  • 经验瓶(80%)(1日) : 4,400 元宝
