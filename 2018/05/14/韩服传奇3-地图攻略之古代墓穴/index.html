韩服传奇3 地图攻略之古代墓穴

翻译：Tekken
原始地址：

※ 最后一次资料更新时间 : 2013年 06月 26日

1. 古墓背景故事

几年前在沙漠中迷路的冒险家无意中找到一个古迹。
进入古迹发现有上百个用泥土做成的士兵和一个巨大的墓碑。
好奇调查发现是一个王的墓碑，继续调查中还发现了一个躲在士兵后面的路。
挪开士兵想继续前进一瞬间，士兵的眼睛突然发光并攻击冒险家。
受伤的冒险家奋力逃出了古墓，并回到村里集合一群人整装待发后重新踏进了沙漠。
但是至今还没找到沙漠中的古墓入口….

2. 古墓简要介绍

  • 推荐给75~85等级的玩家
  • 古墓入口在泰山的<飞沙区域>
  • 古墓中的每一层地图都有进入下一层的游戏玩法
第一层：通过正确输入墓碑中的甲骨文才可进入下一层
第二层：需要猜对固定时间开启的移动中的入口才可以进入下一层
第三层：要开启四个房间中的开关后可通过中间房入口进入最终BOSS房间

3. 古墓进入方法

古墓入口在泰山地图左上侧<飞沙区域（Lv60~80>中找到。
进入<飞沙区域>后往11点方向一直奔跑，可以发现古墓入口。
<飞沙区域>中无法查看小地图和大地图，如第一次进入古墓建议您记一下飞机点。

4. 古墓攻略

古墓第一层攻略

1.第一层介绍

您必须要正确输入古墓石碑中的古代文字<甲骨文>才可以进入下一层

2. 通往第二层的路

  • 必须正确按顺序选中3个古代文字，具体古代文字可完成任务知晓
  • 进入古墓后可接<尸体搜索>任务,完成任务后剩下的二个任务需要打古墓中的士兵获取<盗墓者日记>任务书1和2完成即可。
  • 完成所有的任务后点击墓碑会显示3个正确的古代文字，正确选中3个古代文字后点击[输入完毕]即可移动到第二层
  • 如果随便输入或输入错误，直接传送到<飞沙区域>（任务被重置）

<下图为24个甲骨文组成的暗号示例图>

古墓第二层攻略

1.第二层介绍

第二层怪物较多，您必须正确找到固定时间变化的移动中的入口才可以进入第三层

<下图为2层地图>

2. 通往第三层的路

  • 地图四个边有进入下一层的入口，但开门的时候只有一个入口才能进入第三层
  • 开门的事件和时间可通过系统红字消息查看
  • 入口非总是开放的状态，开门后只有30秒时间才可进入，关闭后需等待下一次开门
作者友情YY：目前开门等待时间约为10分钟，开门时间只有30秒

<下图为入口即将关闭的系统提示>

古墓第三层攻略

1.第三层介绍

您需要打开4个机关后通过中间房中的召唤阵NPC进入BOSS房间

<下图为第三层地图>

2. 通往BOSS房间的路

  • 每一次开关开启，都会刷新大量的怪物
  • 所有的开关开启后，中间房位置会刷新<中BOSS>
  • 击败中BOSS后召唤阵位置上出现石碑NPC，点击NPC即可传送到大BOSS房间
  • 石碑NPC出现后，过一段时间会消失。再次开启，需要重新开启开关

<下图为开关NPC>

<下图为开启所有开关后出现的中BOSS>

<下图为击败中BOSS后出现的石碑NPC>

古墓BOSS房间攻略

1.大王房间介绍

大BOSS-马车 刚开始为沉睡的状态

<下图为BOSS房间地图>

2.击败大BOSS方法

  • 古墓大BOSS位于地图中央
  • 攻打大BOSS过程中如果大BOSS喊话表示激活中BOSS。中BOSS分别以7点，11点，2点方向顺序登场
  • 中BOSS出现时大BOSS为无敌状态，必须首先干掉中BOSS
