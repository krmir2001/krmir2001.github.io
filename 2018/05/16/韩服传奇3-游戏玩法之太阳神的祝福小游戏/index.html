韩服传奇3 游戏玩法之太阳神的祝福小游戏

作者： 泉雨Tekken
原文链接：
http://mir3.mironline.co.kr/info/info_guide_view.asp?p=3cfada52&s=3d1704963d0ffa053d01e4e33cecc5303d08ef74&f=3d1e0f273d0ffa053cf3cfc1&m=3d08ef74

1. 什么是[太阳神的祝福]小游戏？

可以通过进行[太阳神的祝福]小游戏每天可以获得随机BUFF
此小游戏灵感来自于人们在寺庙许愿池扔下硬币保平安中所启发

2. [太阳神的祝福] BUFF获得方法

您可以在每天在泰山城内的太阳女神铜像左右边的水池中仍铜币
如果命中就能获得全天存在的随机BUFF

<下图为泰山城内的太阳女神铜像左右边的水池>

您必须拥有“幸运铜币”才可以开始游戏

幸运铜币

每日一回可以免铜币进行游戏
（4月24日~5月2日为活动期间，可以每日10回免铜币进行游戏） –已结束

<下图为小游戏界面>

上图标注解释

  1. 移动鼠标指针位置可以查看实时变化的信息（距离与角度）
  2. 显示您拥有的幸运铜币数量
  3. 您要投币命中的地方
  4. 力量蓄力条，越往后扔的更远
  5. 扔币可以查看所获得随机BUFF

游戏方法

1. 角度锁定

2. 距离锁定

3. 获得BUFF

  • 一个铜币可以获得一次BUFF
  • 获得共有9种BUFF中的随机BUFF
  • 每个BUFF有Lv1~Lv9等级之分，等级越高BUFF能力值越高
  • 最多可以获得6种不同的BUFF
  • 如果再次扔币所获得重复一样BUFF，留下等级最高的BUFF
  • 所获得BUFF在北京时间11点重置消失
一次性扔多个铜币

  • 默认为一次消耗一个[幸运铜币]
  • 一次消耗10个[幸运铜币]（免费铜币+一般铜币）
  • 一次消耗100个[幸运铜币]（免费铜币+一般铜币）
  • 一次消耗10个[幸运铜币]（高级铜币）
  • 一次消耗20个[幸运铜币]（高级铜币）
// 作者提示：
[免费铜币]指每天给玩家免费送的一次机会
[一般铜币]指元宝商城或寄售中所卖的的[幸运铜币(一般)]
[高级铜币]指[活动期间]商城所卖的[幸运铜币(高级)],作用为大幅度获得BUFF等级

商城中的[幸运铜币] 행운의 동전

4. BUFF种类

BUFF名称 BUFF效果
经验获得量增加 范围：2%~20%
掉落率增加 范围：2%~20%
任务道具掉落率增加 范围：2%~20%
最大HP增加 范围：10~400
最大MP增加 范围：10~400
追加伤害（固定伤害与最高伤害）增加 范围：5~50
死系怪物追加伤害增加 范围：10~90
生系怪物追加伤害增加（对玩家同样适用） 范围：10~90
最终受到伤害减少 范围：1%~20%

另外[官方活动期间]扔币可获得以下BUFF
