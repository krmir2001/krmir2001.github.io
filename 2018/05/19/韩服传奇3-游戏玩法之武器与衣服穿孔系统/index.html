韩服传奇3 游戏玩法之武器与衣服穿孔系统

作者： 泉雨Tekken
原文链接：
http://mir3.mironline.co.kr/info/info_guide_view.asp?p=3cf3cfc1&s=3cfada523d08ef743d1704963d08ef743d01e4e3&f=3d2c24493d01e4e33cf3cfc1&m=3d08ef74

1. 什么是穿孔系统？

穿孔系统指给武器或衣服打孔并镶刻对应的属性石，使您的武器或装备赋予更强大的能力。穿孔系统大体分为”穿孔”,”装备穿孔石”,”穿孔初始化”

// 作者提示：为了更方便大家的理解，定义几个名词
[穿孔] ->下文均解释为 [打孔]
[진동석] 打孔的物品 ->下文均解释为 [打孔石]
[천공석] 上属性的物品 -> 下文均解释为 [属性石]

2. 如何给武器或衣服打孔？

2.1 首先获得[打孔石]

您可以通过钓鱼，打怪获得[打孔石(一般)] [打孔石(高级)] [打孔石(稀有)]

<下图为通过钓鱼获得打孔石示例图>

2.2 诺玛村落-武器店NPC特鲁姆

您可以跟诺玛村落-武器店NPC特鲁姆（167,162）对话，使用打孔系统

<下图为打孔NPC位置小地图>

2.3 窗口中放上武器或衣服和打孔石

  1. 窗口中放上您的武器或衣服

  2. 放上跟装备稀有等级相符合的打孔石

    ~打孔石分类~

    一般打孔石: 可以给 一般 武器或衣服打孔

    高级打孔石: 可以给 高级 武器或衣服打孔

    稀有打孔石: 可以给 稀有 武器或衣服打孔 



// 作者提示 //
打怪获得的[打孔石]，只能给武器或衣服打一次空。
第二个孔，第三个孔需要购买元宝道具[匠人的打孔石]。

匠人的打孔石(장인의진동석)

注意：给武器或衣服最多打孔的数量为3个

<下图为通过打孔石给一般等级衣服打孔示例图>

2.4 打孔完毕后请确认您的装备

  1. 点击开始即可开始打孔
  2. 打孔完成后装备信息下方有灰色字体显示：未上属性石

接下来就可以通过[属性石]物品给已打孔的武器或装备上属性石

3. 如何给已打孔的装备上属性石

3.1 获得[属性石]

您可以通过打怪，任务，制作系统获得各种[属性石]
[属性石]大体分为2类
赤玉系列：给已打孔的武器上属性的石头
青玉系列：给已打孔的衣服上属性的石头

3.2 诺玛村落-武器店NPC特鲁姆 （同打孔NPC）

您可以跟诺玛村落-武器店NPC特鲁姆（167,162）对话，给装备上属性石

<下图为NPC位置小地图>

3.3 放上装备与属性石

  1. 在指定位置放上您要上属性的武器或衣服
  2. 在指定位置放上您的属性石

<下图为给武器上属性石示例图>

注意：给武器或衣服最多可以打3个不同的属性石

3.4 点[开始]给装备上属性-祝您好运

  1. 点击开始按钮给装备上属性
  2. 如果成功，您的装备信息中会显示属性石的能力
注意：装备等级与属性石等级会影响成功概率

<下图为给武器上准确5的示例图>

// 作者提示 //
上属性石需要阶梯性操作，不能直接购买最上级属性石上属性
列如:有些武器准确+7是怎么做到的？
第一次用 [最下级准确属性石] +1
第二次用 [下级准确属性石] +2
第三次用 [中级准确属性石] +3
第四次用 [上级准确属性石] +4
第五次用 [最上级准确属性石] +7

中间任何一个环节失败，有可能影响到其他已上过的属性石会消失。
最坏的结果武器直接被诅咒，具体请看[惩罚机制]

4. 游戏中查看效果

<下图为戴上装备后可在BUFF栏可查看属性石BUFF>

5. 上属性石失败后的惩罚机制

随着您的属性石等级越高，失败概率也会增高。
特别注意：如果上属性石失败时，有可能您的属性石变为诅咒之玉，另外所有之前上过的属性石均会消失。

诅咒之玉：攻击时最小攻击力出现的概率提高

想解除诅咒之玉效果，需要重置属性石

// 作者提示 //
群里大佬提示：[诅咒之玉]效果武器和衣服均有效

6. 重置属性石

6.1 商城购买[活性石]道具


[活性石 활성석]：商城中5500元宝购买此物品，可以重置装备中已上过的所有石头

6.2 找到诺玛村落-武器店NPC特鲁姆 （方式跟上属性石一样）

诺玛村落-武器店NPC特鲁姆（167,162）

<下图为NPC位置小地图>

6.3 开始重置装备中的属性石

注意1：重置后装备已上过的属性石全部消失
注意2：重置后有一定概率获得原先上过的属性石中的一种

<下图为重置属性石方法>

<下图为重置后结果>

7. 属性石分类

同一种属性石分为多个等级

  • 一般属性石：最下级，下级，中级
  • 高级属性石：上级
  • 稀有属性石：最上级
// 作者提示 //
已[准确属性石]为例，同一属性石可分为五个等级
[최하급정확의적옥],[하급정확의적옥],[중급정확의적옥],[상급정확의적옥],[최상급정확의적옥]
分别给武器加 [准确1],[准确2],[准确3],[准确4],[准确7]
以此类推，等级最高加的越多，失败概率越高

<下表为部分属性石种类>

物品名 适用装备 说明
끈기의적옥 武器 +最大体力
인내의적옥 武器 +最大魔力
정확의적옥 武器 +准确
건강의청옥 衣服 +最大体力
정신의청옥 衣服 +最大魔力
바람의청옥 衣服 +敏捷
철벽의청옥 衣服 +最大防御

수호의청옥 衣服 +最大魔防
보호의청옥 衣服 +最小魔防 
// 作者提示 //
除了以上[属性石]外,还有其他+攻速，+技能攻击力等属性石。

8. 属性石合成

可通过诺玛村落-武器店NPC特鲁姆（167,162）合成属性石

注意1：合成根据概率波动有可能会失败，失败时石头全部消失
