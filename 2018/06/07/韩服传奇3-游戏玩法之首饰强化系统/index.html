韩服传奇3 游戏玩法之首饰强化系统

作者： 泉雨Tekken
原文链接：
http://mir3.mironline.co.kr/info/info_guide_view.asp?p=3cfada52&s=3cf3cfc13d1e0f273d2519b83d0ffa053d01e4e3&f=3d01e4e33d08ef743cf3cfc1&m=3d08ef74

1. 什么是首饰强化系统？

全新更新的首饰强化系统替代了之前通过NPC强化的方式，现在只要双击包里的首饰强化石即可强化首饰装备，更方便，更快捷。

2. 首饰强化石及制造材料

物品图片 物品名称 说明
下级首饰强化石(하급장신구제련석) 使用下级首饰强化石强化物品概率为20%
通过制造系统需要2个制练石结晶(제련석결정)打造一个下级首饰强化石(制造需要的材料可以通过打BOSS或元宝道具中获得)
中级首饰强化石(중급장신구제련석) 使用中级首饰强化石强化物品概率为30%
通过制造系统需要2个下级首饰强化石(하급장신구제련석)打造一个中级首饰强化石(制造需要的材料可以通过打BOSS或元宝道具中获得)
最上级首饰强化石(최상급장신구제련석) 使用最上级首饰强化石强化物品概率为100%
此物品只能通过元宝道具中获得
首饰强化保护石(장신구제련수호석) 首饰强化保护石作用为防止强化过程中失败而导致首饰消失
但强化失败导致的强化属性减少是不可避免
此物品单纯是防碎而存在，强化时无论成功与否都会用一次后消失(另外注意：当前首饰附加属性少于2的情况才适用)

※ 只要把首饰制练保护石放在包里，就可以达到目标效果

3. 首饰强化石适用的附加属性

  • 破坏：战士或刺客适用的物理攻击
  • 魔法：也可以称为“全系魔法”，适用于道士或法师的魔法攻击

4. 首饰强化石适用规则

  • 首饰强化成功时：选择的属性 +1 ~ +3 增加
  • 首饰强化失败时：选择的属性 -1 ~ -3 减少

5.首饰强化石使用方法

鼠标左键双击包里的首饰强化石，这时会弹出选择窗。根据提示操作即可

- 5.1 如何使用首饰强化保护石

  • 强化之前您需要装备要强化的首饰
  • 双击包里首饰强化石并选择要强化首饰的位置
  • 如果包里存在首饰强化保护石物品，双击强化石的选择窗口是不一样的

<下图为包里存在首饰强化保护石情况>

<下图为包里没有首饰强化保护石情况>

- 5.2 选择要强化的属性

  • 接下来需要选择要强化的属性
  • 只能选择“破坏”或“魔法”两种
  • 首饰最多可以强化到+10

<下图为选择要强化的属性窗口>

- 5.3 最终确认窗口

  • 此窗口可以看到当前强化目标, 强化属性, 当前数值
  • 强化失败时附加强化属性值 <= 0 则物品会消失

<下图为强化物品最终确认窗>

- 5.4 查看强化结果

  • 强化成功与否可通过聊天窗红字查看
  • 也可以查看装备的属性值是否变化

<下图为强化结果红字显示在聊天窗>

<下图为强化后装备属性发生了变化>

- 5.5 如何制造首饰强化石？

  • 可以通过制造系统制作首饰强化石
  • 制作材料中必须的制练石结晶需要打败BOSS级怪物获得
  • 也可以通过元宝商城中的首饰强化盒中获得
