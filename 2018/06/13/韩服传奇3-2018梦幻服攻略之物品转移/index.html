梦幻服中获得的"部分物品"，您可以转移到"本服"

# 2018年梦幻服新增"可转移物品 - 梦幻精水"
1. 怪物死亡后有可能掉落"梦幻精水盒"
2. "梦幻精水盒"为可转移物品，通过NPC转移到本服
   注意：您只能转移"梦幻精水盒"本身物品
3. 本服中使用"梦幻精水盒"可随机获得"8~12个梦幻精水"
4. "梦幻精水"可在本服中兑换高级稀有物品. 如 "贪狼系列","天鷄系列","麻痹武器系列","高级制造装备"等

1. 如何转移物品？

梦幻服中找到次元移动NPC，保存物品

  • NPC位置在比奇 164,202
  • 可通过NPC保存 黄金币与可转移物品

本服中找到梦幻奖励NPC，取回物品

  • NPC位置在比奇 184,213
  • 可通过NPC取回 黄金币与可转移物品

以下表格为转移条件

物品 梦幻服 本服
黄金币 次数，数量无限制（最多保存1000个） 每个账号每日只能最多取回一次，一次最多50个
可转移物品 只能保管一个种类物品 取回需要30,000元宝
梦幻精水盒 只能保管一个 取回需要30,000元宝

[注意事项]

  • 黄金币每个账号每日只能取回一次，如想增加次数需要花费500元宝/次
  • 可转移物品，只能保存一个种类物品（本服中找回后才能再次保存）
  • 物品保存后无法取消
  • 梦幻精水盒子物品只能在盒子状态才可保存

2. 梦幻精水

1. "梦幻精水盒"物品可转移到本服中
2. 本服中使用"梦幻精水盒"，可随机获得"8~12个 梦幻精水"
3. 本服中可通过"梦幻精水"物品兑换高级,稀有物品. 兑换方式有"固定兑换"和"概率兑换"
    - "固定兑换": 固定数量兑换指定物品
    - "概率兑换": 根据概率兑换指定物品，如兑换失败可获得"精水碎片"物品
4. "精水碎片"物品可通过制造系统制作"神秘梦幻箱子"物品
