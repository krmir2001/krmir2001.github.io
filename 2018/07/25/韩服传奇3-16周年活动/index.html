韩服传奇3 16周年活动

作者： Tekken

最后一次更新日期：2018/07/25

原文链接：http://mir3.mironline.co.kr/news/news_update_view.asp?p=3cf3cfc1&s=3d01e4e33cf3cfc13cecc5303d08ef743d0ffa05&f=3cecc5303d1e0f273cf3cfc13cf3cfc1

1. 活动内容

A. 16周年官网内活动

活动日期：7月25日~8月22日

活动网站：http://event.mironline.co.kr/event/20180725 （需翻墙）

1）积攒16周年点数

积攒方法：

  1. 进入游戏可获得100点（限每日一次）
  2. 击杀BOSS可获得8点（限每日五次）
  3. 16周年幸运宝珠强化可获得50点 （限每日一次）
  4. 完成任务可获得8点 （限每日五次）
  5. 购买元宝物品可获得点数，比例为100元宝可获得1点（无限）

2) 16周年点数兑换游戏物品

// 注意 //
所有兑换的物品不可交易.

3) 16周年点数抽奖

花费300点数抽奖，可随机获得以下物品

4) 查询点数

  1. 打开网站：http://event.mironline.co.kr/event/20180725 （需翻墙，并且是韩国IP）

  2. 登陆

  3. 确认收货角色

  4. 点击按钮兑换

  5. 点击按钮抽奖

5) 活动注意事项

// 注意事项 //
1. 兑换或抽奖后，可在游戏内 "元宝礼物箱" 中获得物品
2. 点数会在活动期间累计，活动结束后清零
3. 只限游戏内元宝商城购买的物品才可获得点数
4. 物品会发放给 "代表角色"，如发放给错误角色不可取消或交换物品
6. 点数兑换的物品不可交换或退货

B. 16周年游戏内活动

活动日期：7月25日~8月22日

1) 强化16周年幸运宝珠活动

7月25日维护后25级以上角色均获得16周年幸运箱，16周年纪念箱

[16周年幸运箱]

  • 只能使用一次
  • 获得经验葫芦300%1小时物品 (♣속성운기(300)(1시간)) 共16个
  • 16周年幸运宝珠
  • 16周年纪念牌
    破坏0-16,全系魔法0-16,幸运1,物防16-16,魔防16-16,黑炎防御0-16,体力/魔力 +250, 经验值20%（活动结束后牌子能力值消失，但穿戴效果依旧存在）

[16周年纪念箱]

  • 24小时内只能使用一次，使用后获得16周年箱

  • 16周年箱：可获得16周年强化石及随机概率性获得其他物品

[16周年幸运宝珠]

由16周年纪念箱开出的16周年箱强化此宝珠物品，满足条件后可到 YD ONLINE NPC 兑换 16周年补给盒（A,B,C）

16周年补给盒（A）

兑换条件：宝珠强化属性为 破坏+16，全系魔法+16

双击盒子后可选择两种兑换方式

第一种：100%获得16周年增益包7天(♣16주년패키지(7일) )**

第二种：随机获得16周年首饰强化石(A,B,C)中一个

16周年补给盒（B）

兑换条件：宝珠强化属性为 破坏+16 或 全系魔法+16

双击盒子后可选择两种兑换方式

第一种：100%获得16周年增益包3天(♣16주년패키지3일) )**

第二种：随机获得16周年首饰强化石(B,C)中一个

16周年补给盒（C）

兑换条件：宝珠强化属性为 破坏+10 或 全系魔法+10

双击盒子后可选择两种兑换方式

第一种：100%获得16周年增益包3天(♣16주년패키지1일) )**

第二种：100%获得16周年首饰强化石(C)一个

注意：16周年补给盒（A,B,C）只能兑换一次，另外16周年装饰强化石 可强化首饰物品

16周年首饰强化石（A）：给首饰强化+5破坏或+3魔法 （最大强化值：+7）

16周年首饰强化石（B）：给首饰强化+3破坏或+3魔法 （最大强化值：+5）

16周年首饰强化石（C）：给首饰强化+2破坏或+2魔法 （最大强化值：+3）

2) 时空副本活动

活动期间所有时空副本进入等级下降并额外获得经验值

副本名称 进入等级（一般） 进入等级（活动时） 经验加成
桃花源 60 30 50%
无尽花园 80 40 50%
长城北部 70 35 50%
长城南部 70 35 50%
沃玛峡穴 1 1 100%
潘夜牛洞副本 1 1 100%
水晶矿山 83 42 50%
深渊棱镜 83 42 50%
鬼灵村 83 42 50%
深渊沃玛神殿 90 45 50%

3) 喊话活动

  • 通过在比奇城内喊话 “ !미르3와 함께 올 여름도 시원하게! ”，可获得凉爽夏日BUFF
  • 喊话时间：北京时间 中午12:00~ 12:30 ， 晚上20:00~20:30
  • 凉爽夏日BUFF： 破坏0-20，全系魔法0-15，物防/魔防 10-17， 体力/魔力 +200，幸运+1，经验值20%

4) “打败闷热天气的方法”活动

游戏内获得烹饪材料，可制作 韩方白饭(한방백숙)，冰冷红豆冰(꽁꽁팥빙수)，冰冷水果冰(꽁꽁과일빙수)

[韩方白饭]
制作材料：鸡肉(닭고기(挖) )， 高丽参(고려인삼)，糯米(햇찹쌀)，黄金栗子(황금밤) //时空副本掉落
功效：物防0-7，魔防0-10，幸运+2，经验值/掉率 +20% 持续时间一小时

[冰冷红豆冰]
制作材料：香甜红豆 1(달콤한단팥 1 NPC 购买), 冰块 2( 얼음 2) //时空副本掉落
增益效果：破坏1~120，全系魔法1~85，追加伤害1~25，准确1~5，敏捷1~5，幸运1~7，致命一击率1~20，致命一击伤害1~10
注意：使用后只能随机获得一个增益效果，效果重复时总是获得最高数值，另外持续时间为1~2小时，增益效果存在时再次使用不会累计持续时间

[冰冷水果冰]
制作材料：香甜水果 1(달콤한 과일 1 NPC 购买), 冰块 2( 얼음 2 ) //时空副本掉落
功效：破坏+30，全系魔法+21，持续时间30分钟

// 特别注意事项 //
* 16周年装饰强化石（A,B,C）物品 活动结束2周后均被删除
* 16周年幸运箱，16周年补给盒，16周年纪念盒，16周年盒，16周年强化石物品活动结束后均被删除
* 冰块，香甜水果，香甜红豆，高丽参，糯米，黄金栗子，韩方白饭物品活动结束后均被删除

5) 元宝物品限时开卖

[16周年增益包 7日]

  • 销售价格：14900元宝
  • 说明：使用后获得 破坏0-16，全系魔法0-13，幸运+2，魔法回避/致命一击概率5% 体力/魔力 +200，舒适+10，经验/掉率+25% 增益效果

[全新齐天大圣时装]

  • 销售价格：11900元宝

// 其他说明 //
* 先前齐天大圣物品均更新最新时装物品
* 时装穿戴效果及特殊技能均不会改变
* 特殊技能为角色下方进度条满后可激活拳头按钮，点击拳头后可释放无视防御的5X5范围技能
