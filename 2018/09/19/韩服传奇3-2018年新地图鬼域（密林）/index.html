0) 背景故事

几百年前，号称密林魔王君临天下的野兽之王跟7人武林江湖高手大战后被封印在此地。

野兽之王手下六角魔兽因主人被封印其力量大大减弱，为了解封自己主人六角魔兽常年守护鬼域（密林）并强化自己力量。

有一天，六角魔兽偶然遇到研究鬼域地区污染原因的武林高手，持续几天大战后最终被高手们击杀。

六角魔兽被消灭后鬼域地区开启了通往被封印的魔王-野兽之王领地的缝隙。

进入次元缝隙后调查队发现被封印的力量因六角魔兽死亡开始慢慢有所减弱。

同时，守护封印魔法的密林人慢慢被魔王精神控制，并开始攻击入侵的调查队武林高手们。

武林高手们意识到这样下去总有一天魔王被解封，并将给玛法大陆造成巨大威胁。

林夏玉看到情况不妙，号召武林高手们重新讨伐魔王。

为了守护密林的和平，请玛法英雄们赶紧上路永久消灭魔王

1）鬼域（密林）介绍

在鬼域地图中消灭六角魔兽后在鬼域坐标（706，72）开启通往鬼域（密林）的缝隙

缝隙持续2小时，进入等级限制90级

鬼域（密林）击杀怪物后很小的概率爆出麻痹戒指和护身戒指碎片

如果六角魔兽在鬼域地图重新刷新，此时密林中的所有角色都会被踢出

4) 如何召唤BOSS-野兽之王

找到A,B,C,D 中的封印魔力石并消灭后野兽之王就会复活

// 温馨提示 //
- 怪物吃道士绿毒跟传染，有可能以后被砍
- 怪物吃法师火墙
- 战士无法群吸
- 最佳练级组合为：法师+道士
- 单练获得经验比90副本-深渊沃玛更高
