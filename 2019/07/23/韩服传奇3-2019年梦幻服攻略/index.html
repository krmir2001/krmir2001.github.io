2019年梦幻服之玩法及奖励介绍

1. 什么是梦幻服？

  1. 梦幻服为限定时间开启的服务器. 在这里您的角色可以更容易，更快速成长起来

    • 梦幻服关闭后角色获得30%经验值转移到本服同名角色
    • 梦幻服升级经验为本服的20%

  2. 梦幻服最多显示四个角色

    • 如果梦幻服中找不到本服中角色，请删除本服中无用角色

  3. 梦幻服关闭后所有梦幻服角色与道具都会重置

    • 未来再次开启梦幻服需要重新来过

2019年梦幻服玩法介绍

  1. 打怪获得的部分道具可转移到本服中

    • 可转移物品包含其额外属性
    • 季节性补给品，现金盒子等物品获得的装备无法转移

  2. 任务立即完成

    • 通过 [任务立即完成券] 或者 [花费元宝]可以立即完成任务
    • 某些特定任务除外（完成奖励交丰富得任务）
    • 100级任务所花费得金币比本服减少到30%

  3. 达成100级之后可参与抽奖

    • 通过抽奖可获得 [魅影之刃] , [麻痹戒指] , [护身戒指]等终极装备
    • 抽一次需花费1万元宝
    • 通过 [额外抽奖券] 可增加抽奖次数

  4. 新增 [特殊制炼石]物品，可无风险升刀

    • 通过 [特殊制炼石]物品升刀，只有 [无变化] 与 [成功] 两种结果

    • 最大制炼Max值为 200

  2. 钓鱼可获得丰厚经验值

      ​ 注意：黄金鳗鱼只能在 [黄金鳗鱼钓鱼点] 才可获得， [黄金鳗鱼钓鱼点] 可通过(比奇NPC 760,669) 进入

      ​ 进入 [黄金鳗鱼钓鱼点] 需要入场券，入场券元宝商店有卖

    • 黄金鳗鱼物品不跟其他经验值加成物品重叠（如经验葫芦，MIRBUFF等）

  3. 梦幻服中特殊NPC销售本服中的部分活动物品

  4. 打怪时根据概率掉落“梦幻精水”物品

    • 梦幻精水物品可转移到本服中，可兑换贪狼系列,天鸡系列,麻痹戒指,高级防具,高级首饰等物品

  5. 部分地区打怪方式平衡性调整

    • 部分怪物等级，刷新率有所调整
    • 部分地区怪物有回位技能，超出范围怪物会回到刷新点
    • 部分地区怪物经验值比本服有所增加

2. 梦幻服特点

  1. 初始角色35级开始

  2. 经验值50%及掉落率30%增益BUFF一直存在

    • 每日一次可获得300%经验葫芦（由于50%经验BUFF一直存在，使用后打怪经验为本服的4.5倍。如本服中稻草人提供10经验，梦幻服中使用300%葫芦后可获得45经验值）

  3. 梦幻服中升级所需要经验为本服的20%

    • 最终本服中所获得经验值为梦幻服获得经验30%，跟等级无关

  4. 购买元宝道具可额外赠送金币

  5. 新建角色后可获得“林夏玉的武林补给箱”，另外部分装备可通过金币购买

- 梦幻服中获得的"部分物品"，您可以转移到"本服"

   注意：您只能转移"梦幻精水盒"本身物品
- 本服中使用"梦幻精水盒"可随机获得"8~12个梦幻精水"

1. 如何转移物品？

梦幻服中找到次元移动NPC，保存物品

  • NPC位置在比奇 164,202

  • 可通过NPC保存 黄金币与可转移物品

本服中找到梦幻奖励NPC，取回物品

  • NPC位置在比奇 184,213

  • 可通过NPC取回 黄金币与可转移物品

可转移物品 只能保管一个种类物品 取回需要30,000元宝

3. 钓鱼方法

- 您必须拥有三种必须道具（"钓鱼服装"，"钓鱼竿"，"钓鱼钩"）才能钓鱼

包含所有自动钓鱼有用的附加属性，另外还包括隐身功能，防止被玩家或怪物攻击

只要您拥有"姜太公的祝福"或"姜太公的贤臣","钓鱼达人的钓鱼竿","钓鱼服装"就可以持续自动钓鱼
