韩服传奇2 2021年夏季狂欢节

活动时间：2021年7月21日 ~ 2021年8月18日 （四周）
原文链接： 夏季活动说明 、7月21日更新说明

活动1：战胜酷暑

（一）全员赠送《充满力量的盒子》

只要玩家在 2021年4月20日 00点 ~ 2021年7月21日 上午9点维护前登录过游戏
都会收到 GM 派发的《充满力量的盒子》

打开盒子后获得以下物品：

  • 夏日玉清水(3日) 破坏/魔法/道术 +1~2, 幸运 +2, 魔防/物防 +2~2
  • 夏日天灵水(3日) 功力+ 3~3, 敏捷+3, 准确+3, 幸运+2, MAXHP+70
  • 夏日经验卷轴(1日)
  • 夏日防御强化酒(3小时)

（二）回归礼物《回归支援盒子》

玩家在 2021年7月21日 ~ 2021年8月18日 回归
赠送 《回归支援盒子》

  • 回归玉清水(3日) x 3个 破坏/魔法/道术 +1~2, 幸运 +2, 魔防/物防 +2~2
  • 回归天灵水(3日) x 3个 功力+ 3~3, 敏捷+3, 准确+3, 幸运+2, MAXHP+70
  • 回归经验卷轴(1日) x 3个
  • 回归防御强化酒(3小时) x 5个

（三）增强组队 Buff

活动期间玩家组队时获得 Buff属性(HP,MP回复之外) x 2倍

（四）无尽白夜

活动期间玛法大陆所有地图均为白天

活动2：夏季避暑方法

（一）(限)酷寒洞穴

活动期间，原来的酷寒洞穴副本变为(限)酷寒洞穴
(限)酷寒洞穴分为三个区域：入口处、酷寒洞穴、贤人的盆地

区域 说明
入口处 刷新大量怪物、杀怪能获得《夏日赠票》
酷寒洞穴 - 分为7个区域，杀怪获得《夏日赠票》
- 限时消灭2~7区域怪时出现《奖励盒子》
贤人的盆地 - 杀怪获得《营养满分鸡肉》《非常完美雪球》《营养满分鸡肉盒》
- 消灭所有贤人的盆地里的怪物后可以通过财管 NPC 召唤最终 BOSS《酷寒千年龟神灵》
  • 在入口处通过跟 酷寒霸主NPC 花费一万金币购买 《瞬间融化的雪球》物品，可以更容易通关副本
  • 限时击杀怪奖励《奖励盒子》可以开出 《夏日赠票》《营养满分肌肉》《十分甜美冰淇淋》《十分酸甜冰淇淋》《酷寒极冰石》《一小块西瓜》
  • 《非常完美雪球》物品最大使用次数为 100回、有效时间为1年
  • 《瞬间融化的雪球》物品最大使用次数为 300回、有效时间为5分钟
  • 《营养满分鸡肉盒》可以开出5个《营养满分鸡肉》和1个《非常完美雪球》
  • 雪球物品为消耗性道具，指定目标后使用时进行范围攻击并拖慢目标的速度

如何获得《韩方参鸡汤》？

  • 在酷寒洞穴杀怪获得 《营养满分鸡肉》
  • 找到活动 NPC，花费10万金币+《营养满分鸡肉》兑换一个《韩方参鸡汤》
  • 《韩方参鸡汤》属性为 破坏/魔法/道术 +20，功力+10，体力/魔力 +200 持续30分
  • 活动 NPC 位于 比奇(337:273) 雪白(300:250) 道馆(188:311) 毒舌山谷(500:481)般若岛(127:335) 盟重(331:323)


《夏日赠票》有什么用？

找到活动 NPC，50个《夏日赠票》可兑换《夏日防具盒》或《夏日首饰盒》

《夏日防具盒》：38种防具中随机获得一个，最好的物品为《天衣无缝》
《夏日首饰盒》：34种首饰物品中随机获得一个，最好的为《金龙的锁链之戒》

如何重复进入(限)酷寒洞穴？

(限)酷寒洞穴每天限进入一次，活动期间在 PC 商城花费 4,400 PC（约27RMB）购买入场卷，可重置一次副本

（二）地狱的魂灵

活动期间玛法大陆出现《地狱的魂灵》怪物，击杀后一定概率掉落《未知铜币》
《未知铜币》使用后随机获得金币（最小1000 最多10万金币），也可以找活动 NPC兑换
20个兑换：祝福油
300个兑换：(限)酷寒洞穴入场卷

// 关于《未知铜币》的金币兑换实验数据 //
使用 1980个 获得 25,778,000 金币
平均 1个 获得 13020金币
假设 1个 祝福油价格为 75000金币（五区为例）
结论 兑换金币价值大于兑换祝福油
