韩服传奇3 2021年中秋活动

翻译：泉雨Tekken
参考资料： 韩文图文攻略

活动时间

2021年9月15日 ~ 2021年10月6日（3周）

活动内容1-击杀贼蚁，获得松饼

  1. 元宝礼物箱获取道具 ♣도둑个미상자（贼蚁盒子）
  2. 双击♣도둑个미상자（贼蚁盒子）获得♣도둑个미소굴(입个권)（贼蚂洞穴（入场卷））
  3. 双击♣도둑个미소굴(입个권)（贼蚂洞穴（入场卷）） 传送到活动地图
  4. 活动地图击杀一般蚂蚁怪物获得♣송편주머니（松饼袋），击杀 boss 怪物获得♣커다란송편바구니（巨大松饼篮）
  5. 袋子递给 比奇 NPC（756,640）获得松饼物品
// 注意事项 //
1. 盒子冷却时间为24小时
2. 入场卷有效时间为2小时，2小时后自动消失
3. 入场卷消失后自动从活动地图踢出
4. 活动地图为一般地图，非副本地图
5. 怪物为活动怪，只能平砍，技能无效
6. 松饼袋可随机从四种松饼中获得一种
6. 巨大松饼篮可获得3个中秋松饼（3小时）

松饼Buff

♣밤송편(栗子松饼) 功效：破坏/全系魔法5-20,物防/魔防4-8 体力/魔力 +200 1小时
♣깨송편(芝麻松饼) 功效：破坏/全系魔法5-25,物防/魔防4-20 体力/魔力 +200 1小时
♣콩송편(大豆松饼) 功效：破坏/全系魔法5-35,物防/魔防4-28 体力/魔力 +250 1小时
♣한가위송편(中秋松饼) 功效：破坏/全系魔法5-50,物防/魔防4-40 体力/魔力 +200 幸运+1 经验值+20% 持续3小时

活动内容2-特别密令

一级密令

  1. 活动期间元宝商店购买 일급비밀임무상자(一级密令盒子)，价格：99,000元宝/个
  2. 双击获得密令卷共五个，打开5个密令卷分别获得1号~5号密令任务
  3. 任务内容分别是击杀四个副本中的最终 BOSS
  4. 完成任务可获得丰厚的奖励（经验，高级技能书页）
  5. 注意：每个角色只能购买一次，并只能做一次相同密令任务
