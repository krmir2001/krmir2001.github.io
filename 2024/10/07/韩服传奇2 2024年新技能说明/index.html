韩服传奇2 2024年新武功说明

1. 新增武功说明

新增武功可以通过以下方式获得:

  • 使用毒药矿山远征队获得的矿山积分兑换
  • 在英雄陵寝异空间、三清英雄陵寝、毒药县中随机获得

每个职业的秘籍最高可达5级，普通武功最高可达4级。
新增武功需要修炼下一级的武功书才能开启新的等级。

  • 秘传可用于0-2级，秘技用于3级，极意用于4级。

武功书构成示例:

  • 狮子吼-秘籍、狮子吼-秘籍:秘传、狮子吼-秘籍:秘技、狮子吼-秘籍:极意

武功开启示例:

  • 狮子吼-秘籍要达到5级，需要1本狮子吼-秘籍，
    另外还需要3本秘传、1本秘技、1本极意。

90级新技能

2. 各职业武功详细说明

[术士武功]

强击-秘籍

魔法攻击武功。作为强击的升级武功，使用时会自动对5格范围内的敌人释放1个落雷（0级基准）。

伤害受魔法和武功等级影响。

修炼值

  • 0级: 74,100 + 强击-秘籍秘传
  • 1级: 125,400 + 强击-秘籍秘传
  • 2级: 192,000 + 强击-秘籍秘传
  • 3级: 338,100 + 强击-秘籍秘技
  • 4级: 545,700 + 强击-秘籍极意

武功升级效果

  1. 对怪物造成15%额外伤害
  2. 25%几率落雷数量翻倍
  3. 对怪物造成30%额外伤害，攻击范围增加到6格
  4. 对怪物造成50%额外伤害，落雷数量增加到2个
  5. 无视敌人魔法抗性，攻击范围增加到7格

吸魔炎风

魔法攻击武功。将强大的火魔之气凝聚成旋风形态，
持续3秒吸引普通怪物（0级基准）。

伤害受魔法和武功等级影响。

修炼值

  • 0级: 3,800 + 吸魔炎风秘传
  • 1级: 6,300 + 吸魔炎风秘传
  • 2级: 9,300 + 吸魔炎风秘传
  • 3级: 15,200 + 吸魔炎风秘技

武功升级效果

  1. 对BOSS怪物造成25%额外伤害
  2. 对BOSS怪物造成50%额外伤害
  3. 对BOSS怪物造成100%额外伤害
  4. 持续时间增加到6秒

吸魔炎风-秘籍

魔法攻击武功。作为吸魔炎风的升级武功，凝聚更大更强的火魔和雷电之气成旋风形态，
在一定时间内吸引普通怪物（0级基准）。

伤害受魔法和武功等级影响。
角色的吸引几率受等级差和魔法抗性影响。

修炼值

  • 0级: 5,400 + 吸魔炎风-秘籍秘传
  • 1级: 8,800 + 吸魔炎风-秘籍秘传
  • 2级: 13,000 + 吸魔炎风-秘籍秘传
  • 3级: 21,600 + 吸魔炎风-秘籍秘技
  • 4级: 31,400 + 吸魔炎风-秘籍极意

武功升级效果

  1. 对BOSS怪物造成100%额外伤害
  2. 对BOSS怪物造成150%额外伤害，持续时间增加到8秒
  3. 可吸引角色，持续时间增加到10秒
  4. 增加爆炸伤害
  5. 对等级较低的敌方角色，每1级差造成5%额外伤害，最多25%
  6. 无视敌人魔法抗性和魔法防御

[战士武功]

狮子吼-秘籍

物理攻击武功。作为狮子吼的升级武功，增加了冷却时间，但可对自身周围5*5范围内的敌人造成伤害，
并使普通怪物和角色分别陷入10秒和1秒的制压状态（0级基准）。

伤害受破坏力和武功等级影响。
制压几率受武功、等级差和目标中毒抗性影响。
制压可被净化术解除，但不受中毒恢复影响。

修炼值

  • 0级: 8,900 + 狮子吼-秘籍秘传
  • 1级: 15,000 + 狮子吼-秘籍秘传
  • 2级: 21,600 + 狮子吼-秘籍秘传
  • 3级: 35,300 + 狮子吼-秘籍秘技
  • 4级: 51,800 + 狮子吼-秘籍极意

武功升级效果

  1. 自身前方目标制压时间翻倍
  2. 对怪物造成100%额外伤害
  3. BOSS怪物制压2秒，普通怪物制压时间增加到20秒
  4. 自身周围攻击范围增加到7*7
  5. 自身前方目标制压时间增加到3倍

极时空神剑

物理攻击武功。以极限境界斩断时空，
贯穿攻击2格范围内的敌人（0级基准）。

伤害受破坏力和武功等级影响。

修炼值

  • 0级: 3,800 + 极时空神剑秘传
  • 1级: 6,300 + 极时空神剑秘传
  • 2级: 9,300 + 极时空神剑秘传
  • 3级: 15,200 + 极时空神剑秘技

武功升级效果

  1. 对怪物造成25%额外伤害
  2. 对怪物造成50%额外伤害
  3. 对怪物造成100%额外伤害
  4. 攻击范围增加到3格

极时空神剑-秘籍

物理攻击武功。作为极时空神剑的升级武功，以极限境界斩断时空，
先贯穿攻击3格范围内的敌人，然后下劈造成范围伤害（0级基准）。

伤害受破坏力和武功等级影响。

修炼值

  • 0级: 5,400 + 极时空神剑-秘籍秘传
  • 1级: 8,800 + 极时空神剑-秘籍秘传
  • 2级: 13,000 + 极时空神剑-秘籍秘传
  • 3级: 21,600 + 极时空神剑-秘籍秘技
  • 4级: 31,400 + 极时空神剑-秘籍极意

武功升级效果

  1. 对怪物造成100%额外伤害
  2. 攻击范围增加到4格
  3. 对怪物造成200%额外伤害
  4. 攻击范围增加到5格
  5. 对等级较低的敌方角色，每1级差造成5%额外伤害，最多25%
  6. 下劈范围攻击附加炎化结效果

[刺客武功]

月下乱舞-秘籍

物理攻击武功。作为月下乱舞的升级武功，增加了冷却时间，但可以更广范围释放剑气并收回，
总共造成2次伤害，并使目标陷入眩晕状态5秒，降低1点精确（0级基准）。

伤害受破坏力和武功等级影响。
眩晕几率受武功、等级差和目标中毒抗性影响。

修炼值

  • 0级: 23,600 + 月下乱舞-秘籍秘传
  • 1级: 38,900 + 月下乱舞-秘籍秘传
  • 2级: 57,600 + 月下乱舞-秘籍秘传
  • 3级: 97,200 + 月下乱舞-秘籍秘技
  • 4级: 148,100 + 月下乱舞-秘籍极意

武功升级效果

  1. 眩晕持续时间增加到6秒，眩晕降低2点精确
  2. 对敌人附加血风击效果
  3. 眩晕持续时间增加到7秒，眩晕降低3点精确，对怪物造成25%额外伤害
  4. 眩晕持续时间增加到10秒，眩晕降低4点精确
  5. 对怪物造成50%额外伤害，眩晕降低5点精确

影华连击

物理攻击武功。将肉身化为影子，暂时不受伤害，
单方面对敌人进行6次连续攻击（0级基准）。

伤害受破坏力和武功等级影响。

修炼值

  • 1级: 7,300 + 影华连击秘传
  • 2级: 10,800 + 影华连击秘传
  • 3级: 17,700 + 影华连击秘技

武功升级效果

  1. 对怪物造成25%额外伤害
  2. 对怪物造成50%额外伤害
  3. 对怪物造成100%额外伤害
  4. 无视敌人防御

影华连击-秘籍

物理攻击武功。作为影华连击的升级武功，将肉身化为影子，暂时不受伤害，
单方面对敌人进行10次连续攻击（0级基准）。

伤害受破坏力和武功等级影响。

修炼值

  • 0级: 6,600 + 影华连击-秘籍秘传
  • 1级: 11,100 + 影华连击-秘籍秘传

  • 4级: 39,900 + 影华连击-秘籍极意

  2. 对怪物造成300%额外伤害
  3. 对怪物造成400%额外伤害
  4. 所有状态异常解除（状态异常中可使用）
  5. 对等级较低的敌方角色，每1级差造成5%额外伤害，最多25%
  6. 影华连击-秘籍的攻击无法回避，攻击范围增加到2格，对怪物造成500%额外伤害

[道士武功]

万效符

投掷具有万种效能的符咒，为自己和队友提供抗魔阵法、大地援护、先天气功、无极真气的效果。

此外，自身受到的伤害减少10%。

伤害减少时间受武功等级影响。

另外，爆杀系得到强化，增加了冷却时间，但可以向范围内的敌人投掷3个符咒。（0级基准）

修炼值

  • 0级: 18,500 + 万效符秘传
  • 1级: 29,900 + 万效符秘传
  • 2级: 43,200 + 万效符秘传
  • 3级: 70,700 + 万效符秘技

武功升级效果

  1. 爆杀系 - 对怪物造成25%额外伤害
  2. 爆杀系 - 对怪物造成50%额外伤害
  3. 爆杀系 - 符咒数量增加到4个
  4. 爆杀系 - 对怪物造成100%额外伤害

万效符-秘籍

作为万效符的升级武功，投掷具有万种效能的符咒，为自己和队友提供抗魔阵法、大地援护、先天气功、无极真气的效果，并施加净化术。此外，自身受到的伤害减少20%。（0级基准）

伤害减少时间受武功等级影响。
另外，爆杀系得到强化，增加了冷却时间，但可以向范围内的敌人投掷4个符咒。

修炼值

  • 0级: 23,600 + 万效符-秘籍秘传
  • 1级: 38,900 + 万效符-秘籍秘传
  • 2级: 57,600 + 万效符-秘籍秘传
  • 3级: 97,200 + 万效符-秘籍秘技
  • 4级: 148,100 + 万效符-秘籍极意

武功升级效果

  1. 爆杀系 - 对怪物造成100%额外伤害
  2. 伤害减少增加25%
  3. 爆杀系 - 符咒数量增加到5个
  4. 爆杀系 - 对怪物造成150%额外伤害，伤害减少增加35%
  5. 爆杀系 - 对等级较低的敌方角色，每1级差造成5%额外伤害，最多25%
  6. 爆杀系 - 符咒数量增加到7个，爆杀系 - 对怪物造成250%额外伤害

血龙手-秘籍

作为血龙手的升级武功，增加了冷却时间，但可以同时召唤所有召唤兽，召唤兽受到的怪物伤害减少25%。（0级基准）

召唤兽的强度受道力和武功等级影响。

修炼值

  • 0级: 23,600 + 血龙手-秘籍秘传
  • 1级: 38,900 + 血龙手-秘籍秘传
  • 2级: 57,600 + 血龙手-秘籍秘传
  • 3级: 97,200 + 血龙手-秘籍秘技
  • 4级: 148,100 + 血龙手-秘籍极意

武功升级效果

  1. 每个召唤兽增加道士的伤害减少3%，召唤兽对怪物造成25%额外伤害
  2. 每个召唤兽增加道士的伤害减少5%
  3. 每个召唤兽增加道士的伤害减少7%
  4. 消耗召唤兽瞬移到召唤兽位置，召唤兽受到的怪物伤害减少50%
  5. 状态异常中可使用，召唤兽对怪物造成50%额外伤害，召唤兽受到的怪物伤害减少75%

[弓手武功]

落雷杀

魔法攻击武功。使用气功在指定位置持续引发落雷。（0级基准）

伤害受破坏力、魔力和武功等级影响。
持续时间受消耗的气功数量影响。

修炼值

  • 2级: 9,300 + 落雷杀秘传
  • 3级: 15,200 + 落雷杀秘技

  2. 对怪物造成50%额外伤害
  3. 对怪物造成75%额外伤害
  4. 对怪物造成100%额外伤害

落雷杀-秘籍

作为落雷杀的升级武功，使用气功在指定位置持续引发落雷。落雷杀使用后10秒内可以对1格范围内的单个敌人追加更强的落雷攻击。（0级基准）

落雷杀的持续时间受消耗的气功数量影响。
